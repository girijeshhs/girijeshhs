Last year, I joined [Plant Green Inertia](https://plantgreeninertia.com) as one of their first engineers. I built a Company Dashboard and a full-stack LMS from scratch — planning to deployment — solo, over 5 months. Both are in production.

Outside of work, I've been building projects around multi-agent AI (Semantic Kernel + Azure OpenAI), computer vision (YOLO), and full-stack web (React, Next.js, FastAPI). A few of them are below.

### Projects

**[SentriChain](https://github.com/girijeshhs)**  
An agentic AI system for querying supply-chain risks in natural language. Uses Semantic Kernel + Azure OpenAI with live Bing Search grounding, Azure SQL & Blob Storage for structured reports, and a Next.js + Tailwind frontend. Built with Prometheus observability baked in.

**[PCB Defect Detection](https://github.com/girijeshhs)**  
Real-time PCB manufacturing defect detection using YOLO. Won 3rd place at the EEC Hackathon.

**[AgriThrive](https://github.com/girijeshhs)**  
A full-stack agriculture platform with crop recommendations, fertilizer guidance, and live commodity prices. Pulls from OpenWeatherMap, SoilGrids, and data.gov.in. Has an in-app AI chatbot for farmers.
